# 🔐 Password Generator

A responsive and customizable **Password Generator** built with **HTML, CSS, Bootstrap, and Vanilla JavaScript**. This application allows users to generate secure random passwords by selecting the desired password length and choosing which character types to include.

## ✨ Features

* Generate secure random passwords
* Choose password length (8–32 characters)
* Include or exclude:

  * ✅ Uppercase letters
  * ✅ Lowercase letters
  * ✅ Numbers
  * ✅ Special characters
* Copy generated password to the clipboard
* Responsive Bootstrap interface
* SweetAlert2 notifications for user feedback
* Modern and clean user interface


---

## 🚀 Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript (ES6)
* Bootstrap 5
* SweetAlert2
* Bootstrap Icons

---

## ⚙️ How It Works

1. Select the desired password length using the slider.
2. Choose which character types should be included.
3. Click **Generate Password**.
4. Copy the generated password with a single click.

If no character type is selected, the application displays an error message prompting the user to select at least one option.

---

## 📋 Character Types

The generator supports the following character sets:

| Type               | Example                                     |
| ------------------ | ------------------------------------------- |
| Uppercase Letters  | A-Z                                         |
| Lowercase Letters  | a-z                                         |
| Numbers            | 0-9                                         |
| Special Characters | ! @ # $ % ^ & * ( ) _ + - = [ ] { } < > ? / |
